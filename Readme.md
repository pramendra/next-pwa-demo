# setup

```
$ git clone https://github.com/pramendra/next-pwa-demo.git
$ cd next-pwa-demo
$ npm install
```

## Run

### Development

```
$ npm run dev
```

### Run Server Side rendering

```
$ npm run build && npm run start
```

### Run client side rendering

```
$ npm run build && npm run export
$ cd out
$ http-server
```
